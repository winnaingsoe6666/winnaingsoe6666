# Code of Conduct

## Purpose

This repository is my main GitHub profile repository. It represents my public technical presence, projects, writing, and collaboration interests.

The goal is to keep discussions professional, constructive, and useful for people interested in machine learning engineering, MLOps, distributed systems, research infrastructure, and related technical work.

## Expected Behavior

Please:

- Be respectful and direct.
- Discuss ideas, code, and technical tradeoffs rather than attacking people.
- Give feedback that is specific, actionable, and grounded in evidence.
- Assume good intent, but ask clarifying questions when something is unclear.
- Respect boundaries around private work, unpublished research, personal information, and security-sensitive details.

## Unacceptable Behavior

The following are not acceptable:

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or behavior.
- Spam, self-promotion unrelated to the repository, or low-effort solicitation.
- Publishing private information without permission.
- Sharing exploit details, secrets, tokens, or security-sensitive information in public issues or discussions.

## Scope

This code of conduct applies to interactions in this repository, including issues, pull requests, discussions, comments, and any direct collaboration that starts from this repository.

## Enforcement

I may edit, hide, delete, or lock comments, issues, discussions, or pull requests that violate this code of conduct. Repeated or severe violations may result in blocking or reporting through GitHub.

If you see behavior that should be addressed, contact me through the channels listed in the README.
